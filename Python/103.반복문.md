# 103. 반복문
반복문은 for, while 문 등이 있는데 사용 방법은 다른 언어들과 유사하다.

> ### for문
* **2차원 배열에 저장할 때**
    ```py
    for i in range(n):
        for j in range(m):
            arr[i][j] = 1
    ```
***
> ### while문
* **일반 조건문**
    ```py
    while i > 0:
        ...
    ```
* **무한 루프**
    ```py
    while 1:
        ...
    ```
