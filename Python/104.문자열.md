# 104. 문자열
문자열 문제가 나왔을 때 자유자재로 문자열을 활용할 수 있어야한다.

> ### 슬라이싱
* **문자열에 저장해서 출력할 때**
    - 문자열을 슬라이싱으로 자를 수 있으며 항상 끝 인덱스는 포함하지 않는다.
    ```py
    s = "helloWorld"
    print(s[:2] + s[2:3] + s[3:])
    ```
* **문자열에 저장해서 출력할 때**
    ```py
    s = "helloWorld"
    for i in s[::-1]:
        print(i)
    ```
